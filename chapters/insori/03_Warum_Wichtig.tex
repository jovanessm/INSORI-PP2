\chapter{Warum ist die Digitalsouveränität wichtig?}
\label{ch:warum}

Die EU haben die DSGVO erstellt, um die Datenschutz ihres Bürgers zu schützen. Gaia-X stellt einen Framework fürs Sicherstellen eines vertrauenswürdigen Data- und Infrastruktursökosystem. Cloud Sovereignty Framework wird erstellt, damit die Cloud Nutzer die von ihren Provider angebotenen Souveränität bewerten können. 

von der "Public Cloud Service Provider Market Leader in Germany" Grafik kann man eindeutig identifizieren, dass die Market Share von Deutschland von US Cloud Hyperscaler (AWS, Microsoft Azure, Google Cloud Platform) dominiert ist. Danach kommt dann die deutschen Cloud Provider wie T-Systems, IONOS und Noris Network.
