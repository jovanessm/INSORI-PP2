\section{Kritische Bewertung der bestehenden Ansätze}
\label{ch:kritische-bewertung}

- Data Act solves Dataportability and Egress Fees, but not application portability (propriotäre Anwendung, wie AWS Lambda), Daten können exportiert, aber Anwendung muss neuentwickelt werden.
- Cloud Act only address nicht personen bezogenen Daten, personenbezogene Daten still falls to dsgvo, which means, that the cloud act widerspruch is still there. Es ändert nichts an der Jurisdiktion über das Mutterunternehmen Der CLOUD Act greift auf Ebene des US-Mutterkonzerns, nicht auf Ebene der EU-Tochtergesellschaft. Solange AWS, Microsoft und Google US-amerikanische Unternehmen sind, unterliegen sie dem CLOUD Act — unabhängig davon, was der Data Act vorschreibt. 
-  Zudem gilt das Egress-Verbot ausschließlich im Kontext eines vollständigen Anbieterwechsels -- laufende Datentransferkosten im Cloud-Betrieb bleiben weiterhin zulässig.
